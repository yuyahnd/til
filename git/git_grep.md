# git grep

git-grep - Print lines matching a pattern

# SYNOPSIS

```bash
git grep [-a | --text] [-I] [--textconv] [-i | --ignore-case] [-w | --word-regexp]
	   [-v | --invert-match] [-h|-H] [--full-name]
	   [-E | --extended-regexp] [-G | --basic-regexp]
       [-P | --perl-regexp]
       [-F | --fixed-strings] [-n | --line-number] [--column]
```
